Color value can be a color name or hexadecimal code.

## Examples

| Description                 | Examples                                          |
|-----------------------------|---------------------------------------------------|
| Color alias (X11 colors)    | `c=Red` `c=DarkRed` `c=Salmon` `c=DarkOliveGreen` |
| Hexadecimal color           | `c=#ff0000`, `c=#32a852`                          |
| Special color compositions  | `c=B&W`                                           |
| Existing image              | `c=I123`                                          |
| User-uploaded image         | `c=U123`, `c=U282833`                             |
| Equally color combinations  | `c=#32a852,#32a852`, `c=Red,DarkRed,#32a852,I123` |
| weighted color combinations | `c=#32a852 5%, #32a852 95%`                       |
